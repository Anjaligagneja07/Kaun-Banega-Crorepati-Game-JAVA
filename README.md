<p align="center">
A desktop-based <b>Kaun Banega Crorepati (KBC) Quiz Game</b> developed using <b>Java Swing, MySQL,</b> and <b>NetBeans IDE.</b> The application provides a complete quiz experience including user registration, login system, player profiles, lifelines, question management, score tracking, prize calculation, and player history.
</p>

---

## 🎯 GAMEPLAY FEATURES
<h3>❓ Random Questions</h3>
<ul>
  <li>Questions fetched dynamically from MySQL database.</li>
  <li>Random question generation.</li>
  <li>Duplicate question prevention.</li>
</ul>

</ul>
<h3>🏆 Score & Prize System</h3>

## 🎁 LIFELINES
The game provides multiple KBC-style lifelines:
<h3>1️⃣ Choose Category</h3>
<ul>
  <li>Select questions from a preferred category.</li>
</ul>
<h3>2️⃣ 50-50 Lifeline</h3>
<ul>
  <li>Removes two incorrect options.</li>
</ul>
<h3>3️⃣ Skip Question</h3>
<ul>
  <li>Skip the current question.</li>
</ul>
<h3>4️⃣ Two Chances</h3>
<ul>
  <li>Get an extra chance if the first answer is wrong.</li>
</ul>

## ⚙️ INSTALLATION GUIDE
**Step 1 :**

Clone the repository :

            git clone https://github.com/Anjaligagneja07/KBC-Game.git

**Step 2 :**

Open the project in NetBeans IDE.

**Step 3 :**

Create a MySQL Database:

            CREATE DATABASE kbc_game;

**Step 4 :**

Import the database tables.

**Step 5 :**

Update MySQL credentials if required:

            Connection con = DriverManager.getConnection( "jdbc:mysql://localhost:3306/kbc_game", "root", "" );

**Step 6 :**

Run the project from NetBeans.

## 🎮 HOW TO PLAY
1. Register a new account.
2. Login using Email and Password.
3. Complete your profile information.
4. Click **Play Game**.
5. Read the instructions.
6. Answer quiz questions.
7. Use lifelines wisely.
8. Earn score and prize money.
9. View your game history.
